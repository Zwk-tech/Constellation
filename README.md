一个专业的心理测评系统，用于评估BDSM倾向中的支配型（Dominant）与顺从型（Submissive）特质。

## 功能特点

- **15道专业问题**：基于心理学研究设计
- **多维度评估**：涵盖15个人格维度
- **5种人格类型**：从强支配型到强顺从型
- **科学评分算法**：精准计算支配-顺从倾向
- **优雅UI设计**：Google Material Design风格
- **隐私保护**：所有数据本地处理，不上传

13. 照料角色 (Care Role)
14. 自主性 (Autonomy)
15. 满足感来源 (Satisfaction Source)

## 人格类型

- **强支配型 (Strong Dominant)** - 75%以上支配倾向
- **偏支配型 (Dominant Leaning)** - 55-75%支配倾向
