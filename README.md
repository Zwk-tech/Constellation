# 专业星座分析系统 (Professional Astrology Analysis System)

一个功能完整的专业星座分析系统，提供太阳/月亮/上升星座计算、12维度深度分析、星座配对和动态运势预测。

## 功能特点

### 1. 精确星座计算
- **太阳星座**：基于出生日期精确计算（必填）
- **月亮星座**：基于VSOP87简化算法计算（选填出生时间）
- **上升星座**：基于球面三角学计算（选填出生时间+地点）
- **城市数据库**：内置20个主要城市的经纬度和时区数据

### 2. 12维度深度分析
每个星座都包含完整的12个维度分析：
- 🎭 性格特质
- 💕 爱情运势
- 💼 事业发展
- 🏥 健康指南
- 💰 财运分析
- 👥 人际关系
- ⭐ 优势潜能
- ⚠️ 劣势警示
- 📈 发展建议
- 🍀 幸运元素
- 💡 适合职业
- 🎯 年度运势

### 3. 星座配对分析
- 智能匹配算法（基于元素、宫位、角度关系）
- 配对分数环形进度图
- 配对等级显示（天作之合/完美匹配/需要磨合/富有挑战）
- 详细分析内容（优势、挑战、建议）
- 支持144种星座配对组合

### 4. 动态运势预测
- 本周运势（基于当前周数）
- 本月运势（基于当前月份）
- 四维运势展示（事业/爱情/财运/健康）
- 确定性随机生成（同一时期内容一致）
- 运势评分和趋势分析

### 5. UI/UX设计
- Material Design 风格
- 紫色渐变主题 (#667eea → #764ba2)
- 星空背景动画
- 卡片展开/收起动画
- 完全响应式设计（支持桌面和移动端）

## 本地使用

直接在浏览器中打开 `index.html` 或 `astrology-analysis.html` 即可使用。

## 使用方法

1. 点击"开始探索"按钮进入测试
2. 填写基础信息：
   - **步骤1**：出生日期（必填）、出生时间和地点（选填）
   - **步骤2**：配对分析（可选）
   - **步骤3**：运势查询（可选）
3. 点击"开始分析"查看详细报告
